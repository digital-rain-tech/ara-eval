### 4. Prompt loading allows path traversal outside `prompts/`
- File path: `labs/lab-01-risk-fingerprinting.py`
- Line number: 95
- Severity: medium
- Category: Diff
- Description: `build_system_prompt()` accepts an arbitrary `rubric` path and passes it to `load_prompt()`, which reads `(PROMPTS_DIR / relative_path)` without verifying that the resolved path stays under `prompts/`. Because `evaluate_scenario()` exposes `rubric` as a parameter, a caller can traverse outside the prompt directory and inject local file contents such as `.env.local` into prompts and request logs.
- Suggested fix: Resolve the candidate path and reject anything outside `PROMPTS_DIR`, or replace free-form paths with an allowlisted rubric registry similar to the jurisdiction/personality loaders.
